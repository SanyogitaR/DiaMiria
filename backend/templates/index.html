DiaMiria

This project demonstrates how machine learning models can be applied in healthcare prediction tasks. It is built using the Pima Indian Diabetes Dataset — a well-known dataset used for binary classification (diabetic / non-diabetic). Please note: this app is an academic project and not a substitute for medical diagnosis.
